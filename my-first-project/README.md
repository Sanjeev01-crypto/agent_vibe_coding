# Welcome to Antigravity! 🚀

This is your workspace. Antigravity is a powerful agentic AI coding assistant designed to pair-program with you, build applications, debug code, and help you research biological or technical systems.

## 🛠️ What We Can Do Together

1. **Web App Development**: Create modern, beautiful, and highly responsive web applications (HTML/CSS/JS, React, Next.js, Vite, etc.) using premium styling and smooth animations.
2. **General Software Engineering**: Write backend code, script automations, build CLI tools, and write tests in Python, Node.js, Go, or any language of your choice.
3. **Scientific & Medical Research**: Query specialized databases (UniProt, PDB, ClinVar, ClinicalTrials.gov, PubMed, etc.) using advanced plugins.
4. **Iterative Design**: Generate mockups, UI elements, or custom assets directly using image generation tools.

---

## ⚡ Useful Slash Commands

Try typing these commands in the chat interface to trigger advanced workflows:
* `/goal` - For complex, long-running tasks where you want me to be extra thorough and not stop until the goal is fully achieved.
* `/grill-me` - Start an interactive interview where I ask you questions to align on design and architecture choices before writing code.
* `/browser` - For tasks requiring web research, browsing documentation, or testing web pages.
* `/schedule` - Schedule a recurring job or set a one-time reminder.
